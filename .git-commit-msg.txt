## 1. Editions 特性驱动验证
- 重写 ProtobufEditionAnnotator，接入 ProtobufFeature 模型
- 基于枚举类型、字段存在性等特性标志驱动验证规则

## 2. 引用解析视觉增强
- 将 4 处 ProblemHighlightType.LIKE_UNKNOWN_SYMBOL 改为 ERROR

- 提升错误可见性，便于开发者快速定位问题

## 3. 字段编号智能推荐
- 重写 FieldNameProvider.nextAvailableFieldNumber()
- 收集同袍字段编号 + reserved 范围 + extension 范围
- 计算最小可用编号（避免与 reserved 冲突）
- 支持向后和向前双向扫描 reserved 声明

## 4. Buf CLI 集成

- buf.yaml 自动发现，输出解析为 IDE 警告注解
- io.kanro.idea.plugin.protobuf-buf.xml 模块配置

## 5. gRPC Reflection 动态发现基础设施
- 新建 grpc/reflection 模块

  - supportsReflection() 健康检查
